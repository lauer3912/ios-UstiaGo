## 📝 App Store 描述 (Description)

```
Clarity - 专注力训练与屏幕时间管理

在数字时代保持专注比以往任何时候都更加困难。Clarity 帮助你了解并管理你的屏幕时间，同时培养健康的专注习惯。

🌟 核心功能

• 专注计时器 - 多种专注模式可选
• 屏幕时间追踪 - 了解您的日常屏幕使用情况
• 专注目标 - 设置每日专注目标并追踪进度
• 成就系统 - 解锁成就，激励持续进步
• 环境音效 - 雨声、海浪、森林等白噪音助您放松
• 数据分析 - 查看专注趋势和洞察
• 健康集成的 - 与 Apple Health 同步
• 晚安模式 - 睡前放松提醒，培养良好作息

🎯 专注模式

• 深度工作 - 25分钟专注
• 创意流动 - 50分钟专注
• 轻松模式 - 15分钟专注
• 迷你冲刺 - 5分钟专注

📊 数据洞察

• 每日/每周/每月专注统计
• 连续专注天数追踪
• 专注时长分布图表
• 与健康数据集成

🏆 成就系统

完成各种挑战解锁成就徽章，追踪您的专注历程。

下载 Clarity，开始掌控您的专注力和时间。
```

**字符数:** ~600 字
